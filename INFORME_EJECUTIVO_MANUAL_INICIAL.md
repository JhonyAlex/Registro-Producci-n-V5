# Informe Ejecutivo + Manual de Uso Inicial

**Sistema:** APP Web de Registro de Producción  
**Dominio:** registro.pigmea.click  
**Fecha:** 16/03/2026

## 1. Resumen Ejecutivo
La APP centraliza el **registro y control de producción por turno** en una plataforma web con acceso seguro, trazabilidad de acciones y análisis operativo.

Su objetivo principal es:
- Capturar datos de producción en tiempo real.
- Facilitar el seguimiento por turno, máquina y operario.
- Entregar visibilidad gerencial mediante dashboard, historial y auditoría.

## 2. Funciones Principales
- **Acceso seguro:** ingreso por código de operario y PIN.
- **Gestión de usuarios:** registro de cuentas y aprobación administrativa.
- **Registro operativo:** carga y edición de datos por turno.
- **Historial inteligente:** filtros por fecha, máquina, jefe y operario.
- **Dashboard ejecutivo:** KPIs, gráficos y exportación.
- **Auditoría:** bitácora de actividades (quién, qué, cuándo).
- **Backups:** exportación/importación de datos para continuidad operativa.

## 3. Qué registra la APP por cada turno
- Fecha
- Turno
- Jefe de turno
- Máquina
- Operario
- Metros producidos
- Cantidad de cambios
- Comentarios / incidencias

## 4. Manual Inicial (Uso Rápido)
1. Ingresar a **registro.pigmea.click**.
2. Iniciar sesión con **código de operario + PIN**.
3. Ir a **Registro** y completar datos del turno.
4. Guardar el registro.
5. Revisar **Historial** para validar, filtrar o editar información.
6. Consultar **Dashboard** para indicadores y análisis.

## 5. Uso para perfiles administrativos
- **Usuarios:** aprobar, desbloquear o eliminar cuentas (según permisos).
- **Actividad:** revisar auditoría de eventos del sistema.
- **Permisos:** ajustar capacidades por rol (administrador).
- **Backup:** exportar antes de cambios críticos e importar cuando sea necesario.

## 6. Estado General
La plataforma está lista para operación diaria y aporta:
- Mejor control de producción por turno.
- Mayor seguridad y gobernanza de accesos.
- Trazabilidad y respaldo de información para gestión y toma de decisiones.
